jQuery Practice Problem Set #2



Hover over the bars to see the animated effect.
Click the bar to take you to the specified site.

    jQuery Problem #1


   Using jQuery, create bars that animate when you hover over them. Make it
   such that when you click on the bars it will take you to the website you
   designate. See above for demontration.
index.html 
  
   <!DOCTYPE html>
   <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.
    min.js"></script>
    <script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8/jquery-
    .min.js"></script>
    </head>
      <body>
      <div id="outline2">
      <button class="yelp"><center>Yelp</center></button><br>
      <button class="cnn"><center>CNN</center></button><br>
      <button class="google"><center>Google</center></button>
      </div><center>Hover over the bars to see the animated effect.<br>
      Click the bar to take you to the specified site.</center>
      </body>
  </html>
style.css 
  .yelp{

     height: 30px;
     width: 100px; 
     background-color: #66CCCC;
     color: white;  
     margin: 0px;
     }

  .cnn{

     height: 30px;
     width: 100px; 
     background-color: red;
     color: white;  
     margin: 0px;
     }

   .google{

     height: 30px;
     width: 100px; 
     background-color: #91bd09;
     color: white;  
     margin: 0px;
  }

   #outline2{
     margin: auto; 
     outline-style:solid;
     outline-color: green;
     margin-top: 50px;
     height: 90px;
     width: 200px;
    }

click.js 
  
 
    $(".yelp").hover(function(){
        $(this).filter(':not(:animated)').animate({ width: "200px" });
        }, function() {
        $(this).animate({ width: "100px" });
        });

        $(".yelp").click(function(){

        window.location.href = "http://www.yelp.com/";

        })

        $(".cnn").hover(function(){
        $(this).filter(':not(:animated)').animate({ width: "200px" });
        }, function() {
        $(this).animate({ width: "100px" });
        });

        $(".cnn").click(function(){

        window.location.href = "http://www.cnn.com";

        })

        $(".google").hover(function(){
        $(this).filter(':not(:animated)').animate({ width: "200px" });
        }, function() {
        $(this).animate({ width: "100px" });
        });

        $(".google").click(function(){

        window.location.href = "https://www.google.com";

        })
Slideshow of images

    jQuery Problem #2


   Using jQuery, produce slideshow as demonstrated above for three images.
index.html 
  <!DOCTYPE html>
  <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.
    min.js"></script>
    <script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8/jquery-
    .min.js"></script>
    </head>
      <body>
      <div id="slideshow">
        <div>
          <img src="ocean1.jpg">
        </div>
        <div>
          <img src="ocean2.jpg">
        </div>
        <div>
          <img src="ocean3.jpg">
        </div>
      </div>
      <center> Slideshow of images  </center> 
      </body>
  </html>
style.css 
  #slideshow {
      margin: 50px auto;
      position: relative;
      width: 260px;
      height: 350px;
      padding: 10px;
      box-shadow: 0 0 20px rgba(0, 0, 0, 0.4);
   }
   #slideshow > div {
      position: absolute;
      top: 10px;
      left: 10px;
      right: 10px;
      bottom: 10px;
    }

   #slideshow img {
      max-width:300px;
      max-height:300px;
      margin-left: 10px;
      margin-top: 10px
  }

click.js 
  
 
    $("#slideshow > div:gt(0)").hide();
    setInterval(function () {
    $('#slideshow > div:first')
        .fadeOut(2000)
        .next()
        .fadeIn(2000)
        .end()
        .appendTo('#slideshow');
        }, 3000);

    Coderbyte Problem 2


 
 Using the JavaScript language, have the function FirstFactorial(num) take 
 the num parameter being passed and return the factorial of it (ie. if num =4,
 return (4 * 3 * 2 * 1)). For the test cases, the range will be between 1 and 18. 


 Input = 4  Output = 24
 Input = 8  Output = 40320
      

  
 function FirstFactorial(num) {

    if (num===0) {

       return 1;}

    else {

       return num*FirstFactorial(num-1);

     }

  })

    Coderbyte Problem 3


 Using the JavaScript language, have the function LongestWord(sen) take the
 sen parameter being passed and return the largest word in the string. If
 there are two or more words that are the same length, return the first word
 from the string with that length.Ignore punctuation and assume sen will not
 be empty. 


 Input = "fun&!! time"  Output = "time"
 Input = "I love dogs"  Output = "love"
      

  
 function LongestWord(sen) { 

   var x=sen.replace(/\W/g," ").split(" ").filter(function(i) {return (i!=="");})

   return x.sort(function(a,b){ return b.length - a.length;})[0]; 

 })

    Coderbyte Problem 4


 Using the JavaScript language, have the function LetterChanges(str) take
 the str parameter being passed and modify it using the following algorithm
 Replace every letter in the string with the letter following it in the
 alphabet (ie. c becomes d, z becomes a). Then capitalize every vowel in
 this new string (a, e, i, o, u) and finally return this modified string. 


 Input = "hello*3"    Output = "Ifmmp*3"
 Input = "fun times!" Output = "gvO Ujnft!"
      

  

  function LetterChanges(str) { 
 
    return str.replace(/[a-zA-Z]/g,function(x) {
    return String.fromCharCode(x.charCodeAt(0)+1); }).replace(/[aeiou]/g,function(y) {
    return y.toUpperCase(); 
    });
  
  }

   Using the JavaScript language, have the function SimpleAdding(num) add up all the numbers 
   from 1 to num.For the test cases, the parameter num will be any number from 1 to 1000.  
 

   Input = 12   Output = 78
   Input = 140  Output = 9870
      

  

  function SimpleAdding(num) { 

    var y=0
    for (var i=1; i<num+1; i++){
       y=y+i;
       }
    return y 
  }

    Coderbyte Problem 6


  Using the JavaScript language, have the function LetterCapitalize(str)
  take the str parameter being passed and capitalize the first letter of
  each word. Words will be separated by only one space.  


  Input = "hello world"  Output = "Hello World"
  Input = "i ran there"  Output = "I Ran There"
      

  

  function LetterCapitalize(str) {
    var c=[]  
    var sep=str.split(" "); 
    for(var i=0; i<sep.length; i++){
    
      c.push(sep[i][0].toUpperCase()+sep[i].slice(1))
    }

    return c.join(" "); 

  }

    Coderbyte Problem 7


  Using the JavaScript language, have the function SimpleSymbols(str) take the
  str parameter being passed and determine if it is an acceptable sequence by
  either returning the string true or false. The str parameter will be
  composed of + and = symbols with several letters between them (ie ++d+===+c++==a) 
  and for the string to be true each letter must be surrounded by a + symbol
  So the string to the left would be false. The string will not be empty and
  will have at least one letter. 


  Input = "+d+=3=+s+"  Output = "true"
  Input = "f++d+"      Output = "false"
      

  

 function SimpleSymbols(str) { 

  var text=str.split("")
  var alpha="abcdefghijklmnopqrstuwxyz"
  
  for (var i=0; i<text.length; i++){
   
    if((alpha.indexOf(text[i])!==-1) && (text[i-1]!== "+" || text[i+1]!== "+"))
     {
          return false;
     }
   }
  
   return true 
         
 }

    Coderbyte Problem 8


  Using the JavaScript language, have the function CheckNums(num1,num2) take 
  both parameters being passed and return the string true if num2 is greater
  than num1, otherwise return the string false. If the parameter values are
  equal to each other then return the string -1. 


  Input: num1=3  & num2 = 122  Output = "true"
  Input: num1=67 & num2 = 67   Output = "-1"
      

  

  function CheckNums(num1,num2) { 

    if(num2 > num1){
    
        return true;
      }
  
    else if( num2 < num1){
        return false;
    }
      
    else 
    {
        return -1;
      }
  }

    Coderbyte Problem 9


  Using the JavaScript language, have the function TimeConvert(num)
  take the num parameter being passed and return the number of hours and 
  minutes the parameter converts to (ie. if num = 63 then the output should
  be 1:3). Separate the number of hours and minutes with a colon. 


  Input = 126  Output = "2:6"
  Input = 45   Output = "0:45"
      

  

    function TimeConvert(num) { 
      h=Math.floor(num/60)
      m=num%60  
      return (h + ":" + m).toString();   
    }

    Coderbyte Problem 10


  Using the JavaScript language, have the function AlphabetSoup(str)
  take the str string parameter being passed and return the string with
  the letters in alphabetical order (ie. hello becomes ehllo). Assume 
  numbers and punctuation symbols will not be included in the string.


  Input = "coderbyte"  Output = "bcdeeorty"
  Input = "hooplah"    Output = "ahhloop"
      

  

  function AlphabetSoup(str) {   
      var x= str.split("") 
      return x.sort().join("") 
  }
